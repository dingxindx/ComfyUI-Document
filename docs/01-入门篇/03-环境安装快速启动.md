# 环境安装与启动（小白重启版）

> 如果你已经按照 `00-小白上路/00-01-安装ComfyUI.md` 完成了安装，可以跳过本章。本章是为需要单独查阅安装步骤的用户准备的精简版。

---

## 一、前提条件速查

| 条件 | 检查命令 | 通过标准 |
|:-----|:---------|:---------|
| Python | `python3 --version` | 显示 3.10 / 3.11 / 3.12 |
| Git | `git --version` | 显示 2.x |
| 空闲硬盘 | `wmic logicaldisk get size,freespace,caption` | 至少 30GB（视频需要 100GB+） |

---

## 二、安装 ComfyUI

```bash
# 进入工作目录
cd ~/workspace

# 克隆
git clone https://github.com/comfyanonymous/ComfyUI.git

# 国内镜像
# git clone https://gitclone.com/github.com/comfyanonymous/ComfyUI.git

cd ComfyUI

# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境（每次启动前都要做！）
source venv/bin/activate   # Linux
# venv\Scripts\activate    # Windows

# 安装 PyTorch（重要——根据显卡选）
# 你的 RTX 4070 Ti Super 需要 CUDA 版本：
# pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# 安装其他依赖
pip install -r requirements.txt

# 国内 pip 镜像
# pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

---

## 三、启动

```bash
# 确保先激活虚拟环境
source ~/workspace/ComfyUI/venv/bin/activate

# 启动
cd ~/workspace/ComfyUI
python main.py
```

看到以下输出说明成功：
```
Total VRAM ... GB, total RAM ... GB
Prompt server running on: http://0.0.0.0:8188
```

浏览器访问 `http://127.0.0.1:8188`。

---

## 四、国内网络环境配置总则

| 操作 | 问题 | 解决方案 |
|:-----|:-----|:---------|
| `git clone` GitHub 仓库 | raw.githubusercontent.com 被限 | 使用 `gitclone.com` 镜像 |
| 从 HuggingFace 下载模型 | 连接超时/速度极慢 | `export HF_ENDPOINT=https://hf-mirror.com` |
| 从 Civitai 下载模型 | 访问不稳定 | 用 哩布哩布 (liblibai.com) 搜同名模型 |
| `pip install` 依赖 | 外网源慢 | `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple` |

---

## 五、启动参数速查

| 参数 | 用途 | 示例 |
|:-----|:-----|:-----|
| `--port` | 改变端口（默认 8188） | `python main.py --port 8189` |
| `--listen` | 允许局域网内其他设备访问 | `python main.py --listen 0.0.0.0` |
| `--novram` | 低显存模式 | `python main.py --novram` |
| `--gpu-only` | 强制只使用 GPU | `python main.py --gpu-only` |
| `--cpu` | 强制只使用 CPU（很慢但兼容） | `python main.py --cpu` |

---

## 六、常见问题快速处理

- **端口被占用** → 用 `--port 8189` 换个端口
- **模型下拉框空白** → 模型没放对目录，检查 `models/checkpoints/`
- **启动报红色错误** → 通常是缺依赖，运行 `pip install -r requirements.txt` 重试
- **ComfyUI 无法访问** → 确认终端还在运行，浏览器地址是 `http://127.0.0.1:8188`
