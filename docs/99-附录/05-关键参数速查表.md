# 关键参数速查表

## 图像生成参数

| 场景 | 节点 | 参数 | 推荐值 | 理由 |
|:----:|:----:|:----:|:------:|:----:|
| 文生图基础 | KSampler | `steps` | 20-30 | SDXL 推荐 20-30 步，超过 40 步回报递减 |
| 文生图基础 | KSampler | `cfg` | 7.0 | SD1.5/SDXL 标准值 |
| ControlNet OpenPose | Apply ControlNet | `strength` | 0.7-1.0 | 姿态控制需要较强信号 |
| ControlNet Canny | Apply ControlNet | `strength` | 0.5-0.9 | 边缘控制影响面广 |
| ControlNet 串联顺序 | Apply ControlNet | 顺序 | OpenPose → Canny | OpenPose 定结构，Canny 定轮廓 |
| IP-Adapter 风格迁移 | IPAdapterApply | `weight` | 0.6-0.8 | 太高复制原图，太低不明显 |
| IP-Adapter + 内容变更 | IPAdapterApply | `weight` | 0.5-0.7 | 保留风格但改变内容 |
| LoRA 堆叠-角色 | Load LoRA #1 | `strength_model` | 0.7-1.0 | 权重最高 |
| LoRA 堆叠-服装 | Load LoRA #2 | `strength_model` | 0.5-0.7 | 权重中等 |
| LoRA 堆叠-风格 | Load LoRA #3 | `strength_model` | 0.3-0.5 | 权重最低 |

## LTX 视频参数

| 场景 | 节点 | 参数 | 推荐值 | 理由 |
|:----:|:----:|:----:|:------:|:----:|
| 文生视频 | LTXVTextToVideoSampler | `steps` | 30-50 | 保证运动连贯性 |
| 文生视频 | LTXVTextToVideoSampler | `cfg` | 3.0-4.0 | 避免运动僵硬 |
| 图生视频 | LTXVImageToVideoSampler | `steps` | 40-60 | 平滑首帧过渡 |
| 图生视频 | LTXVImageToVideoSampler | `image_noise_scale` | 0.1-0.3 | 控制首帧保留程度 |
| 两阶段阶段一 | EmptyLTXVLatentVideo | `width`/`height` | 640×384 | 半分辨率节省显存 |
| 两阶段阶段一 | LTXVTextToVideoSampler | `steps` | 25-35 | 只需确定构图和运动 |
| 两阶段阶段二 | LTXVLatentUpsampler | 放大因子 | 2× | 固定值 |
| 两阶段阶段二 | LTXVTextToVideoSampler | `denoise` | 0.4-0.6 | 细节补充而非重新生成 |
| 蒸馏模式 | LoraLoaderModelOnly | `strength_model` | 0.6-0.8 | 平衡细节和过饱和 |
| 首末帧 | LTXVImageToVideoSampler | `steps` | 50-80 | 保证平滑过渡 |
| 潜空间配置 | EmptyLTXVLatentVideo | `length` | 8×N+1 | 模型步幅约束 |
| 潜空间配置 | EmptyLTXVLatentVideo | `width`/`height` | 32 的倍数 | 模型步幅约束 |

## Wan 2.2 参数

| 场景 | 节点 | 参数 | 推荐值 | 理由 |
|:----:|:----:|:----:|:------:|:----:|
| 标准生成 | ModelSamplingSD3 | `shift` | 4.0 | 影响视频动态幅度和稳定性 |
| I2V 首帧过烤 | LatentCut + LatentConcat | 拼接首帧 | 必须使用 | 消除首帧过度渲染 |
| Lightning | KSampler | `steps` | 4 | Lightning LoRA 仅需 4 步 |
| Lightning | KSampler | `cfg` | 1.0 | Lightning 推荐 |

## TTS 参数

| 场景 | 节点 | 参数 | 推荐值 | 理由 |
|:----:|:----:|:----:|:------:|:----:|
| 语音合成 | ChatTTSNode | `temperature` | 0.3 | 越低越稳定 |
| 语音克隆 | ChatTTSNode | 参考音频 | 3-10秒，清晰无噪 | 质量决定克隆效果 |
| 情感强度 | ChatTTSNode | `emotion_strength` | 0.3-0.6 | 0=中性，0.3-0.6=自然，0.7-1.0=强烈 |
