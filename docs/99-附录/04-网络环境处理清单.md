# 网络环境处理清单

| 操作 | 问题 | 方案 |
|:-----|:-----|:-----|
| `git clone` GitHub | raw.githubusercontent.com 被限 | `git clone https://gitclone.com/github.com/用户名/仓库.git` |
| HuggingFace 下载 | 连接超时 | `export HF_ENDPOINT=https://hf-mirror.com` |
| Civitai 下载 | 访问不稳 | 去 哩布哩布 (liblibai.com) 搜同名模型 |
| pip install | 外网源慢 | `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple` |
| ComfyUI Manager | 默认源在国外 | 配镜像源或手动 git clone |
