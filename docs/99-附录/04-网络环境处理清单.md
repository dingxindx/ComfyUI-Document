# 网络环境处理清单

| 操作 | 问题 | 方案 |
|:-----|:-----|:-----|
| `git clone` GitHub | raw.githubusercontent.com 被限 | `git clone https://gitclone.com/github.com/用户名/仓库.git` |
| | gitclone 镜像 502 | 该镜像已不稳定，备选方案：<br>① `git clone https://gh-proxy.com/https://github.com/用户名/仓库.git`<br>② 挂代理直接 clone<br>③ 去 Gitee 搜索搬运版本<br>④ 用 comfy-cli 方案自动处理 |
| HuggingFace 下载 | 连接超时 | `set HF_ENDPOINT=https://hf-mirror.com`<br>然后下载命令不变，自动走镜像 |
| HuggingFace 模型查看 | 网页打不开 | 用 [hf-mirror.com](https://hf-mirror.com) 浏览和搜索模型 |
| 单独文件下载 | GitHub raw 被墙 | `curl -L https://gh-proxy.com/https://raw.githubusercontent.com/用户名/仓库/main/文件路径`<br>或 `wget https://ghproxy.cherrytwinkle.top/https://raw.githubusercontent.com/...` |
| Civitai 下载 | 访问不稳 | 去 [哩布哩布 (liblibai.com)](https://www.liblibai.com) 搜同名模型；或开代理直连 |
| pip install | 外网源慢 | `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple` 包名 |
| ComfyUI Manager | 默认源在国外 | 配镜像源或手动 git clone |
| conda install | 国外源慢 | `conda install -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/` 包名 |

**GitHub 镜像速查表：**

| 镜像 | 用法 | 状态（2026年5月） |
|:-----|:-----|:----------------:|
| gitclone.com | `git clone https://gitclone.com/github.com/...` | ⚠️ 多次502，不稳定 |
| gh-proxy.com | `git clone https://gh-proxy.com/https://github.com/...` | ✅ 可用，自动跳转至 hub.glowp.xyz |
| ghfast.top | 用于 raw 文件加速 | ✅ 对 raw.githubusercontent.com 有效 |
| Gitee | 搜索搬运版本后 clone | ✅ 稳定但非实时 |
| 代理/VPN | `git clone https://github.com/...` | ✅ 最可靠 |

> **建议优先级**：gh-proxy.com → gitclone.com（失败时切换）→ 代理直连 → Gitee
