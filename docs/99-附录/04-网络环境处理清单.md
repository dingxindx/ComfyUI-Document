# 网络环境处理清单

## GitHub 访问

| 操作/资源 | 问题类型 | 处理方案 |
|:---------:|:--------:|:--------|
| `git clone` GitHub 仓库 | raw.githubusercontent.com 被限 | 使用 `gitclone.com` 镜像：`git clone https://gitclone.com/github.com/用户名/仓库.git` |
| 下载 GitHub 仓库 ZIP | 下载慢/失败 | 在 哩布哩布 搜索同名节点下载压缩包 |
| ComfyUI Manager 安装插件 | 默认源在国外 | 方法1：Manager 设置中配置镜像源；方法2：手动 git clone 镜像地址；方法3：下载压缩包解压到 `custom_nodes/` |

## 模型下载

| 操作/资源 | 问题类型 | 处理方案 |
|:---------:|:--------:|:--------|
| 从 HuggingFace 下载模型 | 连接超时/速度慢 | 设置 `export HF_ENDPOINT=https://hf-mirror.com`，或直接访问 `hf-mirror.com` |
| 从 HuggingFace 下载大模型（>10GB） | 下载中断 | 使用 IDM/aria2 多线程下载；使用 hf-mirror.com；或从 哩布哩布 搜索同名模型 |
| 从 Civitai 下载模型/LoRA | 访问不稳定 | 在 哩布哩布 搜索同名模型；或使用浏览器 + 下载工具（IDM/aria2）|
| 访问 Civitai 图片元数据 | 需要代理环境 | 使用 哩布哩布 图片（部分包含元数据）|
| Gemma 3 12B 文本编码器 | 大文件下载 | 替代方案：使用内置 T5 编码器 |
| Wan 2.2 模型下载（5B/14B） | 大文件下载 | 优先使用 hf-mirror.com；14B 需 30GB+ 硬盘空间 |

## Python 包管理

| 操作/资源 | 问题类型 | 处理方案 |
|:---------:|:--------:|:--------|
| `pip install -r requirements.txt` | 外网源慢 | 使用 PyPI 镜像：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple` |
| 自动下载的模型（ChatTTS 等） | 首次运行卡住 | 手动从镜像站下载后放入对应目录，跳过自动下载 |

## 重要提示

- HuggingFace 镜像站：`export HF_ENDPOINT=https://hf-mirror.com`
- GitHub 镜像：`git clone https://gitclone.com/github.com/...`
- PyPI 镜像：`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple`
- 国内模型平台：哩布哩布 AI (liblibai.com)
