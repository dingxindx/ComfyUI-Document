## 🎯 学习路径推荐

### 新手入门路线
```
00-小白上路/准备工作 → 安装 → 界面认知 → 首次生成
          ↓
01-入门篇/核心概念 → Manager 安装
          ↓
02-图像生成篇/文生图工作流 → ControlNet → IP-Adapter → LoRA
```

### 视频生成路线
```
03-视频生成-LTX/核心概念 → 文生视频 → 图生视频 → 两阶段
          ↓
04-视频生成-Wan → 05-语音合成
```

### 进阶
```
06-学习路径方法  →  深入理解每个参数
99-附录/速查表  →  随时查阅
```
