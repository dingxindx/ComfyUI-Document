## 🎯 推荐学习路径

### 新手入门路线（第 1-3 天）

```
00-小白上路/准备工作 → 安装 → 界面认知 → 首次生成
          ↓
01-入门篇/核心概念 → Manager 安装
          ↓
02-图像生成篇/文生图工作流 → 社区资源获取
```

### 图像控制路线（第 4-7 天）

```
02-图像生成篇/ControlNet → IP-Adapter → LoRA堆叠
          ↓
图生图工作流 → 局部重绘Inpaint → 图像放大
          ↓
Flux 文生图工作流（质变——转入新一代架构）
```

### 视频生成路线（第 2 周）

```
03-视频生成-LTX/核心概念 → 文生视频 → 图生视频 → 两阶段
          ↓
04-视频生成-Wan（全文生图→图生视频→Lightning→进阶）
          ↓
05-视频后处理/帧插值与超分 → 06-语音合成
```

### 日常参考

```
07-学习路径方法  →  理解工作流设计哲学
99-附录/速查表  →  随时查阅
00-小白上路/04-故障排查手册  →  遇到问题先翻这里
```
