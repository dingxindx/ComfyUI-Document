## 🎯 学习路径推荐

### 新手入门路线

```
入门概念 → 环境搭建 → 跑通文生图 → ControlNet → IP-Adapter → LoRA
```

### 视频生成路线

```
LTX 基础概念 → 文生视频 → 图生视频 → 两阶段高质量 → 首末帧控制
```

### 进阶路线

```
Wan 2.2 视频 → TTS 语音 → 低显存优化 → 参数调优
```
