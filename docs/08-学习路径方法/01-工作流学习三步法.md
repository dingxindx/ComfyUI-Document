# 工作流学习三步法

把你想要的效果拆解成"能力模块 → 节点 → 模型"三步。

## 第一步：输出决定输入

先不要想节点，问：画面里什么是不可妥协的？

| 核心要求 | 能力模块 | 对应技术 |
|:---------|:---------|:---------|
| 固定角色脸/物 | 身份保持 | IPAdapter / InstantID |
| 特定画风 | 风格控制 | LoRA / Style Model |
| 精确姿态 | 骨架或深度控制 | ControlNet (OpenPose/Depth) |
| 多元素融合 | 区域融合 | Attention Mask / LayerDiffuse |
| 换背景 | 重绘/抠图 | Inpaint / SAM / RemoveBG |

练习：列 5 个你想做的工作流，填这 张表。

## 第二步：节点 = 能力模块的"标准插头"

不要背节点，按功能分类记：

| 能力 | 节点名 |
|:-----|:-------|
| 提示词 | CLIP Text Encode (Prompt) |
| ControlNet 预处理 | Auxiliary Preprocessors |
| 身份保持 | IPAdapter Unified Loader + Apply |
| 重绘 | VAE Encode (for Inpaint) |
| 姿态控制 | OpenPose Preprocessor |

关键心法：90% 的工作流只用 10 个核心节点。看颜色连线（紫=模型，蓝=潜空间，橙=条件）比记名快 3 倍。

## 第三步：模型选择——抄作业

1. 去 Civitai/哩布哩布搜你想做的效果
2. 看 Recipe，直接列出 Base Model + LoRA + CFG + Steps
3. **复制这套组合**
