# 五天脚手架练习法

每天练一种信息流模式，从模板改起。

## 第 1 天：文生图入门
- 拖入自带 `sd1.5` 范例工作流
- 把 checkpoint 换成 DreamShaper 8
- 增加一个 LoRA 节点
- 重点学：CLIP 编码 vs Conditioning 的含义

## 第 2 天：ControlNet
- 找一张人像照片，保持姿势换外貌
- 安装 `ComfyUI-Impact-Pack`
- 拖入 OpenPose + ControlNet 工作流
- **常见卡点**：预处理器下载失败 → 手动下载放 `models/controlnet/`

## 第 3 天：IP-Adapter 换脸/换衣
- 安装 `ComfyUI_IPAdapter_plus`
- 用 sd1.5 版（模型小，国内可下载）
- 练到：把一张图的人脸迁移到新图片上

## 第 4 天：区域绘制与修复
- 圈出眼睛区域只改眼睛
- 必用节点：VAE Encode (for Inpaint) + Set Latent Noise Mask
- 理解 Mask 和 Latent 噪声的关系

## 第 5 天：成品反推
- 在平台找一张带完整参数的图
- 只看 Model/LoRA/CFG/Steps/Seed
- 手动复现每一个节点，不用模板
- 差异在哪？controlnet 缺失还是 VAE 不对？

## 最快速上手路径

1. 拖入自带工作流 → 换 DreamShaper → 加 LoRA → 跑通
2. 加 ControlNet(OpenPose)
3. **出错一次，解决一个红框错误**
4. 3 天后，你开始能预判需要什么节点
