# 社区资源获取指南

## 一、三大模型社区速览

| 平台 | 特点 | 获取方式 | 典型用途 |
|------|------|---------|---------|
| **HuggingFace** | 最全的基座模型 + 官方权重 | `git lfs` / 浏览器下载 | 基础模型、ControlNet、IP-Adapter |
| **Civitai** | 社区微调模型 + LoRA 最丰富 | 浏览器下载 / API下载 | LoRA、微调 Checkpoint、风格模型 |
| **哩布哩布 AI** | 国内平台，高速下载 | 浏览器直接下载 | 国内用户首选，大部分 Civitai 模型有搬运 |

## 二、从 Civitai 获取参数还原工作流

### 步骤 1：获取参数信息

点击 Civitai 图片下方的 "Show Metadata" → "Download JSON" / "Download Image with Metadata"。

元数据中包含：
- `prompt` / `negative_prompt`
- `steps`, `sampler_name`, `cfg`
- `seed`
- `model`（checkpoint 名称 + 下载链接）
- 所有 LoRA 及其权重

### 步骤 2：在 ComfyUI 中搭建工作流

**最简单的方法**：下载图片（含元数据）→ 将 PNG 文件**直接拖入 ComfyUI 画布**。ComfyUI 会自动读取嵌入的工作流 JSON 并还原节点图。

### 步骤 3：采样器名称对照

| Civitai 显示 | ComfyUI sampler_name | ComfyUI scheduler |
|-------------|:--------------------:|:-----------------:|
| DPM++ 2M Karras | `dpmpp_2m` | `karras` |
| Euler a | `euler` | `ancestral` |
| DPM++ 2M SDE Karras | `dpmpp_2m_sde` | `karras` |
| Euler | `euler` | `normal` |

## 三、模型搜索技巧

1. 去 Civitai 或 哩布哩布 搜你想做的**效果图的关键词**
2. 看该图的 **"Recipe"** 区域，会直接列出：Base Model + LoRA + CFG + Steps
3. **复制那套组合**，不要自己猜
