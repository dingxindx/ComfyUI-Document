# 社区资源获取指南

## 一、三大模型社区对比

| 平台 | 特点 | 国内访问 | 用途 |
|:-----|:------|:---------|:-----|
| **HuggingFace** (huggingface.co) | 最全基座模型，官方权重 | 需镜像（hf-mirror.com）| 基础模型、ControlNet、IP-Adapter |
| **Civitai** (civitai.com) | 社区微调、LoRA 最丰富 | 需代理 | LoRA、风格模型、微调 Checkpoint |
| **哩布哩布** (liblibai.com) | 国内平台，高速下载 | 直接访问 | 国内首选，大部分 Civitai 模型被搬运 |

## 二、从 Civitai 获取图片参数

1. 打开 Civitai 目标图片 → 点击 "Show Metadata"
2. 元数据中包含：prompt、negative_prompt、steps、CFG、seed、model、LoRA 列表
3. **最简单方法**：下载图片 → **直接拖入 ComfyUI 画布** → 自动还原工作流

### 采样器名称对照

| Civitai 显示 | ComfyUI sampler_name | ComfyUI scheduler |
|:-------------|:--------------------:|:-----------------:|
| DPM++ 2M Karras | dpmpp_2m | karras |
| Euler a | euler | ancestral |
| DPM++ 2M SDE Karras | dpmpp_2m_sde | karras |
| Euler | euler | normal |

## 三、模型搜索技巧

1. 去 Civitai/哩布哩布 搜你想做效果的关键词
2. 看该图的 Recipe 区域，会列出 Base Model + LoRA + CFG + Steps
3. **复制那套组合**，不要自己猜参数

## 四、国内下载策略速查

| 资源 | 方案 |
|:-----|:-----|
| HuggingFace 模型 | `export HF_ENDPOINT=https://hf-mirror.com` |
| GitHub 节点 | `git clone https://gitclone.com/github.com/...` |
| Civitai LoRA | 去 哩布哩布 搜同名 |
| pip 依赖 | `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple` |
