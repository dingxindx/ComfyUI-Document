# 文生图基础工作流

这是复现 Civitai 图片参数的标准工作流。适用于任何文生图场景。

## 一、整体结构

```
Load Checkpoint → CLIP Text Encode (Prompt)
                → CLIP Text Encode (Negative)
                → Empty Latent Image
                → KSampler → VAE Decode → Save Image
```

## 二、逐节点搭建

### 节点 1：加载 Checkpoint（Load Checkpoint）

| 项目 | 说明 |
|------|------|
| **添加方式** | 右键 → 搜索 "Load Checkpoint" → 或从菜单 `add_node → loaders → Load Checkpoint` |
| **添加位置** | 画布最左侧 |
| **输入连接** | 无（根节点） |
| **关键参数** | `ckpt_name`: 选择下载的 checkpoint |
| **输出连接** | `MODEL` → KSampler；`CLIP` → CLIP Text Encode；`VAE` → VAE Decode |

### 节点 2：CLIP 文本编码器 - 正面提示词

| 项目 | 说明 |
|------|------|
| **添加方式** | 右键 → 搜索 "CLIP Text Encode" |
| **输入连接** | `clip` ← Load Checkpoint 的 `CLIP` |
| **关键参数** | `text`: 从 Civitai 元数据复制的 positive prompt |
| **输出连接** | `CONDITIONING` → KSampler 的 `positive` |

### 节点 3：CLIP 文本编码器 - 负面提示词

同上，`text` 写入 negative prompt，`CONDITIONING` → KSampler 的 `negative`。

### 节点 4：Empty Latent Image

| 项目 | 说明 |
|------|------|
| **添加方式** | 右键 → 搜索 "Empty Latent Image" |
| **输入连接** | 无（根节点） |
| **关键参数** | `width`/`height`: 目标分辨率；`batch_size`: 1 |
| **输出连接** | `LATENT` → KSampler 的 `latent_image` |

### 节点 5：KSampler

| 项目 | 说明 |
|------|------|
| **添加方式** | 右键 → 搜索 "KSampler" |
| **输入连接** | `model`/`positive`/`negative`/`latent_image` |
| **关键参数** | `seed`（从元数据复制或 -1 随机）、`steps`（20-30）、`cfg`（7.0）、`sampler_name`（如 `euler`）、`scheduler`（如 `karras`）、`denoise`（1.0） |
| **输出连接** | `LATENT` → VAE Decode |

> 💡 Civitai 的采样器名称与 ComfyUI 对照：
> - Civitai "DPM++ 2M Karras" → ComfyUI: `sampler_name=dpmpp_2m, scheduler=karras`
> - Civitai "Euler a" → ComfyUI: `sampler_name=euler, scheduler=ancestral`

### 节点 6：VAE 解码（VAE Decode）

| 项目 | 说明 |
|------|------|
| **输入连接** | `samples` ← KSampler；`vae` ← Load Checkpoint |
| **输出连接** | `IMAGE` → Save Image |

### 节点 7：保存图像（Save Image）

| 项目 | 说明 |
|------|------|
| **输入连接** | `images` ← VAE Decode |
| **参数** | `filename_prefix`: 自定义前缀 |

## 三、从 Civitai 获取参数并还原

1. 打开 Civitai 目标图片 → 点击 "Show Metadata" → "Download JSON" / "Download Image with Metadata"
2. 将 PNG 文件**直接拖入 ComfyUI 画布** — ComfyUI 会自动读取嵌入的工作流 JSON 并还原节点图
3. 若无法自动还原，手动按上述模板搭建
