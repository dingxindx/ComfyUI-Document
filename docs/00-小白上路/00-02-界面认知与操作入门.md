# 第一次打开 ComfyUI —— 界面认知与操作入门

请按上一章步骤启动 ComfyUI 后，在浏览器打开 `http://127.0.0.1:8188`。如果你的终端只显示了 `ComfyUI` 文件夹而没有运行 `python main.py`，请先回到上一章完成启动。

---

## 一、你看到的界面——共分为 5 个区域

```
┌─────────────────────────────────────────────────────────┐
│ [菜单栏] File  Edit  View  Queue  Workflow  Canvas  ...│
├─────────────────────────────────────────────────────────┤
│                                                         │
│                                                         │
│   [画布区域] —— 这里是主战场，所有节点搭在这里            │
│                                                         │
│       ┌──────────┐      ┌──────────┐                     │
│       │ 节点 A   │──────│ 节点 B   │                     │
│       │          │      │          │                     │
│       └──────────┘      └──────────┘                     │
│                                                         │
│                                                         │
├─────────────────────────────────────────────────────────┤
│ [+] Queue Prompt  [×] Interrupt     |   [状态栏]         │
└─────────────────────────────────────────────────────────┘
```

### 区域 1：顶部菜单栏

最上面一行文字（File / Edit / View 等）。大部分功能用鼠标右键就能完成，菜单栏主要是辅助作用。

### 区域 2：画布（Canvas）—— 核心区域

整个灰色的空白区域。这是你的"工作台"，所有节点都摆在这里。

### 区域 3：侧边栏（可能是右侧或左侧）

默认可能不显示。如果看到竖排的小图标，那是快捷工具栏。

### 区域 4：底部按钮

**最重要的两个按钮**：
- `[+] Queue Prompt`（添加到队列）—— 单击这个按钮开始生成
- `[×] Interrupt`（中断）—— 生成过程中点这个停止

### 区域 5：右下角状态栏

显示当前的运行状态信息。

## 二、鼠标基本操作

> 💡 **核心心法**：ComfyUI 的画布操作逻辑和普通的图形设计软件（如 Photoshop、Figma）非常相似。如果你用过任何一个，会很快上手。

### 2.1 右键添加节点

**在画布空白处 → 单击鼠标右键**，会弹出一个搜索菜单：

```
┌──────────────────────────────┐
│ Search...                     │
├──────────────────────────────┤
│ checkpoint                    │
│ ├ CheckpointLoader            │
│ ├ Load Checkpoint             │
│ └ ...                         │
│ latent                        │
│ ├ Empty Latent Image          │
│ └ ...                         │
│ ...                           │
└──────────────────────────────┘
```

**使用方法**：
1. 右键 → 输入框出现
2. 输入你想加的内容的关键词（如 "checkpoint"、"KSampler"）
3. 下拉列表会实时过滤
4. 点击你要的节点 → 它就出现在画布上了

> ✅ **试试看**：在画布空白处右键，输入 `checkpoint`，选择 `CheckpointLoaderSimple` 看看效果。

### 2.2 移动节点

- **鼠标左键按住节点标题栏**（顶部深色区域）→ 拖动 → 松开放置
- 移动到其他节点附近时，ComfyUI 会自动吸附对齐（磁吸效果）

### 2.3 连线和断线

**连线**：鼠标左键按住节点的**圆形输出端口**（在节点右侧），拖出一条线，连接到另一个节点的**输入端口**（在节点左侧），松开。

```
┌──────────┐         ┌──────────┐
│ 节点 A   │──●────●→│ 节点 B   │
│          │   连线   │          │
└──────────┘         └──────────┘
    输出端口             输入端口
```

> 💡 端口有不同的颜色——每种颜色代表一种数据类型。**紫色**是模型、**蓝色**是潜空间张量、**橙色**是条件、**粉色**是 CLIP 文本编码器。

**断线**：
- 方法 1：按住 `Ctrl`，点击连线 → 删除
- 方法 2：点击选中连线（变白色高亮），按 `Delete` 或 `Backspace` 键

### 2.4 选中和框选

- **单击**一个节点 → 选中它（边框变亮）
- **在画布空白处按住左键拖动** → 出现一个框（矩形选区）→ 松开 → 框内的所有节点被选中
- 选中多个节点后可以一起拖动

### 2.5 删除节点

选中节点 → 按 `Delete` 或 `Backspace` 键

### 2.6 缩放和移动画布

| 操作 | 效果 |
|:----|:----|
| 鼠标滚轮 | 放大/缩小画布 |
| 按住鼠标中键拖动 | 平移画布（就像拖动地图）|
| 按住 Ctrl + 滚轮 | 放大/缩小 |
| 空格键按住 + 左键拖动 | 平移画布（另一种方式）|

### 2.7 双击空白处

双击画布空白处也可以弹出搜索框添加节点。

## 三、认识节点的"长相"

所有节点有相同的组成部分：

```
┌──────────────────────────────┐ ←── 标题栏（深色背景，显示节点名称）
│ KSampler                     │
│ ┌──────────────────────────┐ │
│ │ seed:           -1     ─┤│ │ ←── 参数输入区域（可以修改数值或文字）
│ │ steps:           20     ─┤│ │
│ │ cfg:              7.0   ─┤│ │
│ │ sampler_name:  euler  ──┤│ │
│ │ scheduler:    normal  ──┤│ │
│ └──────────────────────────┘ │
│ ● model          ● LATENT   │ ←── 端口（左侧=输入，右侧=输出）
│ ● positive                   │
│ ● negative                   │
│ ● latent_image               │
└──────────────────────────────┘
```

### 端口类型

| 端口位置 | 含义 | 样子 |
|:--------|:-----|:-----|
| 节点左侧 | **输入端**——接收数据 | 一个圆点，有标签名 |
| 节点右侧 | **输出端**——发出数据 | 一个圆点，有标签名 |

### 端口颜色快速参考

| 颜色 | 数据类型 | 例子 |
|:----:|:---------|:-----|
| 🟣 紫色 | MODEL（模型） | Checkpoint 的模型输出 |
| 🔵 蓝色 | LATENT（潜空间） | KSampler 的潜空间输出 |
| 🟠 橙色 | CONDITIONING（条件） | CLIP Text Encode 输出 |
| 🩷 粉色 | CLIP（文本编码器） | CLIP 模型参数 |
| 🟢 绿色 | IMAGE（图像） | VAE Decode 输出的图像 |
| 🟡 黄色 | VAE（变分自编码器） | VAE 模型参数 |
| 🔴 红色 | MASK（遮罩） | 用于重绘 |

## 四、参数输入的操作方法

### 数字参数

直接点击数字 → 出现光标 → 输入新数字

或者：鼠标放到数字上 → 左右拖动 → 快速调整（类似音量滑块）

### 下拉选择

点击显示当前选项的框 → 弹出下拉列表 → 选择

常见下拉参数：`sampler_name`、`scheduler`、`ckpt_name` 等

### 文本参数（提示词）

点击大文本框 → 输入文字 → 点击其他地方完成输入

## 五、第一次生成——运行一个已存在的工作流

ComfyUI 安装后自带一个示例工作流（如果是全新安装的话）：

### 方法 1：加载自带模板

1. 点击顶部菜单栏 `Workflow → Browse Workflow Templates`
2. 弹出窗口中会列出各种模板
3. 选择 "Text to Image (SD1.5)" 或类似的文生图模板
4. 点击加载

> ⚠️ **注意**：如果你是新安装的 ComfyUI，还没有下载任何模型，加载模板后节点会显示红色（表示模型文件找不到）。这很正常。下一章教你下载模型。

### 方法 2：从文件加载工作流

如果有人给了你一个 `.json` 工作流文件：
1. 点击菜单 `Workflow → Load`
2. 选择 JSON 文件
3. 或者直接把 `.json` 文件拖入浏览器窗口

## 六、常犯错误和心理建设

### 常见错误

| 错误 | 后果 | 怎么避免 |
|:-----|:-----|:---------|
| 输入端口接了多根线 | 只有最后一根有效，其他被覆盖 | 一个输入端口只能接一根线 |
| 端口类型不匹配 | 连不上——线条会变红 | 同颜色端口才能连接 |
| 参数没设置（默认为空） | 生成失败或效果极差 | 检查每个节点的参数 |
| 忘记激活虚拟环境 | 终端找不到 Python | 启动前检查 `(venv)` 前缀 |
| 模型放错目录 | 节点下拉框里找不到模型 | 确认 `models/` 下的子目录正确 |

### 心理建设

**"出错是正常的，每一个红框都是一次学习机会。"**

- 节点变红：表示缺少该节点的依赖或模型
- 线条变红：表示连接的数据类型不匹配
- 生成全黑/全绿：通常是 VAE 连接问题
- 生成花了很长时间：对，这就是 AI 的正常速度
