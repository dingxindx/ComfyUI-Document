# 准备工作：检查你的电脑能不能用 ComfyUI

在开始安装之前，先花 5 分钟确认你的电脑是否满足条件。这一步可以避免后续遇到各种"装上了跑不动"的问题。

---

## 一、你的显卡（最重要！）

ComfyUI 的 AI 计算严重依赖显卡。**没有好显卡也能用，但会非常慢**。

### 怎么查自己的显卡？

**Windows 用户：**
1. 键盘按下 `Win + R`，输入 `dxdiag`，回车
2. 弹出的窗口"显示"选项卡，看"名称"一栏
3. 或：右键桌面 → 显示设置 → 高级显示 → 显示适配器属性

**Windows 用户（就是你）：**
1. 键盘按下 `Win + R`，输入 `dxdiag`，回车
2. 查看弹出窗口的"显示"选项卡中的"名称"一栏
3. 你的显卡是 **NVIDIA RTX 4070 Ti Super（16GB VRAM）**，完全满足 ComfyUI 所有需求

**Linux 用户：**
```bash
lspci | grep VGA
# 或
nvidia-smi   # 如果是 NVIDIA 显卡
```

### 你的显卡能做什么？

| 显卡类型 | 显存 | 能做什么 | 推荐模型 |
|:--------:|:----:|:---------|:---------|
| **NVIDIA RTX 4090/4080/4070** | 12-24GB | ✅ 全部流畅运行 | SDXL、Flux、LTX视频 |
| **NVIDIA RTX 3060/4060** | 8-12GB | ✅ 文生图、简单视频 | SD1.5、SDXL、LTX fp8 |
| **NVIDIA GTX 1060/1660** | 6GB | ⚠️ 文生图可用 | SD1.5 模型 |
| **Intel/AMD 集成显卡** | 无独立显存 | ❌ 不建议使用 | — |

### 确认显卡驱动已安装

**NVIDIA 用户**：在终端输入 `nvidia-smi`，看到类似下表的输出表示驱动正常：
```
+-----------------------------------------------------------------------------+
| NVIDIA-SMI 535.xxx    Driver Version: 535.xxx    CUDA Version: 12.2        |
+-----------------------------------------------------------------------------+
```

如果提示"command not found" → 需要安装 NVIDIA 驱动。

## 二、安装 Python（编程语言环境）

ComfyUI 是用 Python 写的，电脑上需要有 Python 解释器。

### 怎么查看是否已安装？

打开终端（CMD 或 PowerShell），输入：
```bash
python --version
# 或
python --version
```

### 你看到的结果

| 显示结果 | 含义 | 怎么办 |
|:---------|:-----|:-------|
| `Python 3.10.x` / `3.11.x` / `3.12.x` | ✅ 已安装，版本符合 | 跳过安装步骤 |
| `Python 3.8.x` / 3.9.x | ⚠️ 版本太低 | 升级到 3.10+ |
| `command not found` | ❌ 未安装 | 需要安装 |

> 💡 ComfyUI 推荐 Python 3.10 - 3.12。**不建议使用 3.13**，部分节点可能不兼容。

### 安装 Python（如果需要）

**Windows 用户**：
1. 前往 [python.org](https://www.python.org/downloads/)
2. 下载 Python 3.11 或 3.12
3. 安装时**务必勾选** "Add Python to PATH"

---

## 三、安装 Git（版本管理工具）

Git 用来从 GitHub 下载 ComfyUI 代码。

### 检查是否已安装
```bash
git --version
```

如果能显示 `git version 2.x.x`，说明已安装。没有的话前往 https://git-scm.com/download/win 下载安装。

---

## 四、检查硬盘空间

ComfyUI 本身很小（几百 MB），但**模型文件非常大**：

| 模型类型 | 大小 | 存放位置 |
|:---------|:----:|:---------|
| SD1.5 checkpoints | ~2-5GB | `models/checkpoints/` |
| SDXL checkpoints | ~7GB | 同上 |
| LTX 2.3 fp8 | ~23GB | 同上 |
| Wan 2.2 5B | ~10GB | 同上 |
| ControlNet | ~1.5GB/个 | `models/controlnet/` |
| LoRA | 30-150MB/个 | `models/loras/` |
| VAE | 300-800MB | `models/vae/` |

**建议至少预留 50GB 空闲硬盘空间**（如果你打算玩视频生成，建议 100GB+）。

### 查看硬盘剩余空间
```bash
# Windows
wmic logicaldisk get size,freespace,caption
```

## 五、网络环境检查

> ⚠️ **墙内用户重点关注**：ComfyUI 的代码托管在 GitHub，模型文件托管在 HuggingFace。这些平台在国内可能无法直接访问或速度极慢。

### 测试 GitHub 连通性
```bash
ping github.com
```
如果长时间无响应或丢包严重，说明需要镜像或代理。

### 准备好镜像工具
书中所有需要下载的地方，都会提供国内镜像方案。常见的镜像包括：

| 原始地址 | 国内镜像 |
|:---------|:---------|
| `github.com` | `gitclone.com` |
| `huggingface.co` | `hf-mirror.com` |
| `pypi.org` | `pypi.tuna.tsinghua.edu.cn` |
| `civitai.com` | `liblibai.com`（哩布哩布）|

---

## 检查清单

在进入下一步之前，确认以下条件：

- [ ] 我知道自己的显卡型号和显存大小
- [ ] Python 3.10+ 已安装（终端输入 `python --version` 确认）
- [ ] Git 已安装（终端输入 `git --version` 确认）
- [ ] 硬盘至少有 30GB 空闲空间（玩视频需要 100GB+）
- [ ] 我已经知道如何访问 GitHub（直接连接或使用镜像）

如果以上都 ✅，恭喜！你的电脑可以开始装 ComfyUI 了。
