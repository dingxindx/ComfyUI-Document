# LTX 2.3 两阶段高质量工作流

这是 LTX 2.3 的**精髓工作流**。核心思想：先低分辨率确定构图和运动，再潜空间放大补充细节。

## 一、两阶段流程图

```
阶段一（半分辨率 640×384, 25-35步）
CheckpointLoader → LTXVTextToVideoSampler
                                    ↓（latent 输出）
阶段二（潜空间 2× 放大 + 精炼）
LatentUpscaleModelLoader → LTXVLatentUpsampler（1280×768）
LoraLoaderModelOnly（Distilled LoRA）
LTXVTextToVideoSampler（15-20步, denoise 0.4-0.6）
                                    ↓
VAEDecodeTiled → SaveVideo
```

## 二、阶段一参数

| 节点 | 参数 | 值 | 理由 |
|:-----|:-----|:--:|:-----|
| EmptyLTXVLatentVideo | width | 640 | 半分辨率 |
| | height | 384 | 半分辨率 |
| | length | 33 | — |
| KSampler | steps | 25-35 | 只需确定构图和运动 |
| | cfg | 4.0-5.0 | 略高以确定运动方向 |

## 三、阶段二：潜空间放大

### 加载 Distilled LoRA（可选）

| 节点 | 参数 | 值 |
|:-----|:-----|:---|
| LoraLoaderModelOnly | lora_name | `ltx-2.3-22b-distilled-lora-384` |
| | strength_model | 0.6-0.8 |

### 放大节点

| 节点 | 参数 | 值 |
|:-----|:-----|:---|
| LatentUpscaleModelLoader | model_name | `ltx-2.3-spatial-upscaler-x2-1.0` |
| LTXVLatentUpsampler | width | 1280（2×）|
| | height | 768（2×）|

### 精炼采样

| 参数 | 值 | 说明 |
|:-----|:--:|:-----|
| steps | 15-20 | 精炼阶段不需太多步 |
| cfg | 3.0-4.0 | 与阶段一保持一致或略低 |
| denoise | 0.4-0.6 | **关键**：保留已有结构的同时增加细节 |

> **denoise 含义**：0.4 表示保留 60% 的第一阶段结构，补充 40% 的新细节。不是"重新生成"。

## 四、场景参数速查

| 场景 | 阶段一步数 | 阶段二步数 | denoise | LoRA 强度 |
|:-----|:---------:|:---------:|:-------:|:---------:|
| 人像特写 | 30 | 20 | 0.5 | 0.7 |
| 风景/场景 | 25 | 15 | 0.6 | 0.5 |
| 动作/运动 | 35 | 18 | 0.4 | 0.6 |
| 风格化/艺术 | 25 | 15 | 0.5 | 0.8 |
