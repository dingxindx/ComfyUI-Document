# LTX 2.3——环境准备与模型部署

## 一、安装自定义节点

```bash
cd /d %USERPROFILE%\workspace\ComfyUI\custom_nodes\
git clone https://gitclone.com/github.com/Lightricks/ComfyUI-LTXVideo.git
```

**如果还需要 GGUF 量化模型支持**（低显存方案）：
```bash
git clone https://gitclone.com/github.com/city96/ComfyUI-GGUF.git
```

安装后重启 ComfyUI。

## 二、下载模型

以下模型全部从 HuggingFace 下载，使用 hf-mirror.com 镜像：

```bash
set HF_ENDPOINT=https://hf-mirror.com
```

| 模型 | HuggingFace 仓库 | 放到哪 |
|:-----|:-----------------|:-------|
| LTX 2.3 fp8 | Lightricks/LTX-2.3-22B-fp8 | `models/checkpoints/` |
| Gemma 3 12B fp4 | Lightricks/gemma_3_12B_it_fp4_mixed | `models/text_encoders/` |
| 潜空间放大模型 | Lightricks/LTX-2.3-spatial-upscaler-x2 | `models/latent_upscale_models/` |
| Distilled LoRA | Lightricks/LTX-2.3-22B-distilled-lora-384 | `models/loras/` |
| LTX Audio VAE (可选) | Lightricks/LTX-2.3-audio-vae | `models/vae/` |


## 四、Workflow Templates（最快捷方式）

安装好节点和模型后：
1. 菜单栏 → **Workflow** → **Browse Workflow Templates**
2. 搜索 "LTX-2.3"
3. 加载模板（Text to Video / Image to Video）
4. 若弹窗提示缺少模型，手动从 hf-mirror.com 下载后放入对应目录
5. 点击 Refresh → 不会再次弹窗
