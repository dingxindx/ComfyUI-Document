# 环境准备与模型部署（LTX 2.3）

## 一、必需的自定义节点

### 节点 1：ComfyUI-LTXVideo（官方核心节点）

| 项目 | 说明 |
|------|------|
| **GitHub** | https://github.com/Lightricks/ComfyUI-LTXVideo |
| **安装方式** | ComfyUI Manager → 搜索 "LTXVideo" → Install → 重启 |
| **手动安装** | 下载 ZIP → 解压到 `custom_nodes/ComfyUI-LTXVideo-master` → 重命名去掉 `-master` |

### 节点 2：ComfyUI-GGUF（可选，GGUF 量化模型使用）

| 项目 | 说明 |
|------|------|
| **GitHub** | https://github.com/city96/ComfyUI-GGUF |
| **安装方式** | ComfyUI Manager → 搜索 "GGUF" → Install → 重启 |

## 二、模型下载清单

| 模型 | 来源 | 存放路径 |
|------|:----:|---------|
| `ltx-2.3-22b-dev-fp8.safetensors`（推荐 16-24GB） | HuggingFace | `models/checkpoints/` |
| `ltx-2.3-22b-dev.safetensors`（24GB+） | HuggingFace | `models/checkpoints/` |
| `gemma_3_12B_it_fp4_mixed.safetensors` | HuggingFace | `models/text_encoders/` |
| `ltx-2.3-spatial-upscaler-x2-1.0.safetensors` | HuggingFace | `models/latent_upscale_models/` |
| `ltx-2.3-22b-distilled-lora-384.safetensors` | HuggingFace | `models/loras/` |
| `ltx-audio-vae.safetensors`（可选） | HuggingFace | `models/vae/` |

> ⚠️ 网络提示：使用 `hf-mirror.com` 镜像站下载，或从 哩布哩布 AI 搜索 "LTX 2.3" 搬运资源。

## 三、文件结构验证

完成下载后，目录结构应为：

```
ComfyUI/
├── models/
│   ├── checkpoints/
│   │   └── ltx-2.3-22b-dev-fp8.safetensors
│   ├── text_encoders/
│   │   └── gemma_3_12B_it_fp4_mixed.safetensors
│   ├── latent_upscale_models/
│   │   └── ltx-2.3-spatial-upscaler-x2-1.0.safetensors
│   ├── loras/
│   │   └── ltx-2.3-22b-distilled-lora-384.safetensors
│   └── vae/
│       └── ltx-audio-vae.safetensors (可选)
└── custom_nodes/
    ├── ComfyUI-LTXVideo/
    └── ComfyUI-GGUF/ (可选)
```
