# LTX 2.3 图生视频工作流

从一张静态图片生成运动视频。

## 一、与文生视频的区别

| 差异点 | 文生视频 | 图生视频 |
|:-------|:---------|:---------|
| 采样器 | `LTXVTextToVideoSampler` | **`LTXVImageToVideoSampler`** |
| 额外节点 | 无 | `LoadImage` + `ImageResize` |
| 提示词重点 | 描述场景+运动 | **只描述运动**（场景在图片里）|

## 二、新增节点

### LoadImage + ImageResize

加载参考图并缩放到与 EmptyLatentVideo 一致的分辨率：

- `width`/`height`: 与 EmptyLatent 相同（如 832×480）
- `resampling`: lanczos
- 输出 → 连接采样器的 `image` 端口

### LTXVImageToVideoSampler

| 参数 | 推荐值 | 说明 |
|:-----|:------:|:-----|
| `steps` | 40-60 | 比文生视频更多步，平滑首帧过渡 |
| `cfg` | 3.0-3.5 | 过高会扭曲原图 |
| `image_noise_scale` | 0.1-0.3 | 控制首帧保留程度；越低越接近原图 |

## 三、提示词技巧

**错误**：❌ "a beautiful mountain landscape"（描述场景）
**正确**：✅ "camera slowly pans left, clouds drift across the sky, light rays through trees"（只描述运动）

## 四、检查清单

- [ ] 参考图片分辨率与目标分辨率一致或成比例
- [ ] 图片已被 ImageResize 调整到正确尺寸
- [ ] 提示词主要描述运动而非场景
- [ ] `image_noise_scale` 在 0.1-0.3 范围内
- [ ] steps 在 40-60 范围内
